# Student Registration Form

## Overview

The Student Registration Form is a responsive web application developed using HTML and CSS. It is designed to collect student information through a clean and user-friendly interface. The project demonstrates front-end development skills, including form design, layout management, and responsive styling.

## Features

* Student information registration form
* Responsive design for different screen sizes
* Clean and organized user interface
* Input fields for personal and academic details
* Form validation using HTML attributes

## Technologies Used

* HTML5
* CSS3
